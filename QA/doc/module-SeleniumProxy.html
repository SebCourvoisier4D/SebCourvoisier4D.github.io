Module: SeleniumProxy

The SeleniumProxy module is is meant to be used as a singleton by the BuildRunner module in order to instantiate an Express HTTP server that publishes the resources required by Selenium on port 9100. It's a singleton because there could be only one Express instance at once so don't instantiate it yourself. This module requires the modules https://github.com/strongloop/expressjs.com, https://github.com/jprichardson/node-fs-extra, https://nodejs.org/api/path.html and https://nodejs.org/api/http.html
