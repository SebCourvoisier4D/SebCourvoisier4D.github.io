Module: TestRunner

The TestRunner module is dedicated to run a test. This module requires the modules https://github.com/jprichardson/node-fs-extra, https://nodejs.org/api/path.html, https://nodejs.org/api/os.html, https://github.com/caolan/async, https://github.com/isaacs/node-glob, https://github.com/arturadib/shelljs and https://github.com/Leonidas-from-XIV/node-xml2js

(static) enableRemoteHandler() → {module:TestRunner}

Enable the HTTP remote handler that allows an external process to register a PID to kill (GET http://127.0.0.1:9080/addPID/<pid>), to unregister a PID (GET http://127.0.0.1:9080/removePID/<pid>) and to send a "heartbeat" (GET http://127.0.0.1:9080/heartbeat). The HTTP remote handler either returns "done" or "denied".
