Module: Config

The Config module is dedicated to the configuration of the test machine. It handles everything related to the global and local settings (loading, getting, setting...). This module requires the modules https://nodejs.org/api/fs.html, https://nodejs.org/api/path.html, https://nodejs.org/api/os.html, https://github.com/isaacs/node-glob, https://github.com/caolan/async and https://github.com/npm/osenv
