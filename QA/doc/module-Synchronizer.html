Module: Synchronizer

The Synchronizer module is dedicated to syncing the tests with Perforce. This module requires the modules https://nodejs.org/api/fs.html, https://nodejs.org/api/path.html, https://nodejs.org/api/os.html, https://github.com/caolan/async, https://github.com/isaacs/node-glob and https://github.com/arturadib/shelljs
